快捷键（Shortcut of MacOs）

---

Android Studio：

```none
Android Studio 快捷键
完成现在的声明：opt+cmd+enter 
减小字体大小：ctrl+减号
重置字体大小：ctrl+等于
删除行：ctrl+delete 
删除单词到开始、结尾：cmd(opt)+delete、fn+delete
移动到行尾、首：fn+左、右，带选择的+shift、osx系统级：ctrl+e、a
移到上、下一个单词：cmd(opt)+左、右
向上、下滑动，必要时移动：cmd+上、下
当前行后、前插入行：cmd+enter、+shift+enter
找到类、文件：cmd+o、shift+o
文件结构：ctrl+shift+o
向上、下移动声明：cmd+shift+上、下
向上、下移动行：opt+shift+上、下
复制行：cmd+d
前进后退：cmd+左右
找到高亮错误：cmd+’
显示错误描述：ctrl+‘
项目结构：cmd+;
代码上下文菜单：ctrl+;
自动补全：cmd+/
优化导入：ctrl+opt+o
新建，辅助生成：ctrl+enter(cmd+n)
上下文操作行为：alt+enter
打开选中项的上下文菜单：ctrl+shift+\
override方法：ctrl+o
打开声明：cmd+b，ctrl+shift+下
多行编辑选择：opt+shift+click
打开动作action：cmd+shift+a
目标选择：ctrl + \，在 finder 里面打开
代码围绕补充：cmd+opt+t
inspection 检查：cmd+opt+shift + i /删除未使用的资源（unused resources）
列选择模式：cmd+shift+8
代码 surround with: opt+cmd+t
重构 Refactor this: ctrl + t
多重编辑器 前、后一个 tab：ctrl+shift+左、右
编辑器前、后一个 tab：ctrl+左、右
前、后一个方法：ctrl+上、下
```



Terminal：

```yml
操作：
	段落首/末：ctrl+a/e
	下/上个词：opt+右/左

	删除游标后的第一字符：ctrl+d
	删除游标前/后所有内容（行）：ctrl+u/k
	删除游标前的一个单词：ctrl+w
	删除游标后的一个单词：opt+d(preference, use option as meta key：Command-Option-O)
	
	调换前后字符：ctrl+t
	选中整行：点三下
	选中单词：点单词
	选中矩形块：opt+拖动（单纯的拖动是复选行）
	拷贝纯文本：opt+shift+cmd+c
	不带背景色拷贝：ctrl+shift+cmd+c
	选中 url/dir：shift+cmd+双击
	打开 url：cmd+双击
	反向搜索命令历史：ctrl+r
	
	（标记：输入命令的行，书签：特殊的行，可以用于定位）
	上/下一个标记：cmd+上/下
	插入书签：shift+cmd+m
	插入带名字的书签：opt+shift+cmd+m
	跳转上/下个书签：opt+cmd+上/下
	清除到上一个标记：cmd+l
	清除到上一个书签：opt+cmd+l（会在书签之前停下来）
	
窗口：
	拆分面板：cmd+d （拆分面板只是为了比较内容用的，上下内容会一致）
	关闭拆分面板：cmd+shift+d
```
